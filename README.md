# Super Wacky PC Builder

Welcome to the **Super Wacky PC Builder**! This is a modern full-stack web application built as our semester project for **CP4485: Emerging Trends**. 

The goal of this site is to give gamers and hardware enthusiasts a clean, easy-to-use interface for planning out their next custom computer build without having to rely on messy spreadsheets or endless digging through forum threads.

## 🚀 The Tech Stack

This project is built using current industry tools and workflows, focusing on performance and responsive design:

* **Frontend Framework:** React with Next.js (App Router)
* **Styling:** Tailwind CSS (supporting dark mode)
* **Language:** JavaScript

## 📂 Current Features (Milestone 1)

For the first milestone, we focused heavily on setting up a responsive frontend, component-based development, and clean navigation:

* **Consistent Navigation:** Uses Next.js `layout.js` to keep the experience seamless across the site.
* **Reusable UI Components:** Features dedicated modular components, like our unified `<Footer />` and `<Navbar />`, to prevent duplicate code.
* **Multi-Page Structure:** Includes distinct pages for a **Home** landing experience, a core **System Builder** page, an **About** page detailing why we built this, and initial **Parts** and **Contact** pages.
* **Responsive Grids:** Fully optimized layout using Tailwind so the hardware lists stack cleanly on mobile screens and expand beautifully on desktop resolutions.

## 🔮 Upcoming Features (Planned for Later This Semester)

As the semester progresses, we will be expanding this into a fully operational full-stack application by adding:

* **Database Integration:** Storing component data dynamically and allowing users to save their builds.
* **User Authentication:** Secure OAuth login so users can access their saved configuration profiles from any device.
* **AI Service Integration:** An intelligent system checkup feature that hooks into an AI API to automatically scan a user's drafted loadout and warn them about potential hardware bottlenecks or mismatched parts.
